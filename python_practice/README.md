## 학습내용

### Python 기초 문법
코드를 직접 짜는 것보다 읽고 흐름을 파악하는 수준이 목표입니다.

| 구분 | 내용 | 수준 |
|---|---|---|
| 필수 | 변수 / 조건문 / 반복문 / 함수(def) | 코드 흐름 읽기 |
| 필수 | 클래스 / self / `__init__` | AI 코드 전체가 클래스 구조 |
| 필수 | 리스트 / 딕셔너리 | 데이터 담는 기본 구조 |
| 권장 | 리스트 컴프리헨션 | AI 코드에 자주 등장 |

### 라이브러리
메서드를 외우는 것이 아니라 패턴을 익히는 것이 목표입니다. 논문 코드 분석하며 자연스럽게 체화합니다.

| 라이브러리 | 용도 | 익혀야 할 수준 |
|---|---|---|
| NumPy | 배열(텐서) 다루기 | `shape` / `reshape` / 슬라이싱 패턴 |
| PyTorch | AI 모델 구현 | `forward` / `loss` / `optimizer` 4줄 패턴 |
| Pandas | 데이터 전처리 | CSV 읽고 정제하는 기본 사용법 |
| HuggingFace Transformers | 모델 불러와 실행 시키기 | `pipeline`으로 5줄 안에 실행하기 |
| LangChain | RAG 파이프라인 | 튜토리얼 따라 RAG 챗봇 만들기 |
| Gradio | 모델에 UI 입혀서 데모 배포 | `Interface` 함수로 입출력 연결 |
