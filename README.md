# AI-study

AI 스터디 기록 저장소입니다.

## 소개
이 저장소는 AI 관련 논문 정리와 Python 기초 코드 연습을 기록하기 위해 만들었습니다.

## 스터디 목표
- 핵심 논문 완독 및 코드 리딩 역량 향상
- Python 기초 익히기 및 결과물 GitHub 업로드
- 논문을 읽으며 코드 구현 병행 후 결과물 GitHub 및 Hugging Face 업로드
- Hugging Face에서 모델을 불러와 실행하고 Gradio 데모 제작

 ## 운영 구조

### 매일
- Python 문제풀이 30분 및 업로드
- Python 기초 300제 또는 민코딩 등에서 자기 수준에 맞는 문제 풀이
- Colab 노트북으로 풀이하고, 코드에 풀이 과정 주석 작성
- GitHub 업로드 후 단톡방에 링크 공유
- 하루 30분 내외로 짧고 간단하게 진행


### 화·수·목 — 개인 자율 학습 (권장 2시간)
- 해당 주 논문 읽기
- 논문 기반 코드 구현
- 결과물 GitHub 및 Hugging Face 업로드
- 주 1회 이상 코드 구조 공부 및 GitHub / Hugging Face 활용 연습

※ 요일은 화·수·목이 아니어도 조율 가능  
※ 단, 금요일까지 GitHub 및 Hugging Face 업로드 완료

### 금요일 — 주간 점검 (30분)
- 수업 후 짧게 모여 이번 주 공부 현황 공유
- 주말 스터디 전 어려운 부분 1차 공유
- 주말 발표 준비 상태 점검

## 학습 방식
- 논문 읽기와 코드 구현을 병행
- 완벽한 구현보다 구조 이해와 실습 경험을 우선
- 학습 결과물은 꾸준히 업로드하여 기록

## 공부 목표
- AI 논문을 읽고 핵심 내용을 정리할 수 있기
- Python 기초 문법에 익숙해지기
- 논문 기반 코드를 읽고 이해할 수 있기
- Hugging Face와 Gradio를 활용해 간단한 데모를 만들 수 있기
